demo-repo

A tiny demo repo to show a GitHub-like layout built with HTML & CSS.

This demonstrates a repository view: file list + README preview + quick actions. Use this template to showcase your projects without backend — it's purely static.

How to use

  • Open the single HTML file in a browser to preview the UI.
  • Edit the file list, change project name, or pin repos in the left panel.
Tip: I can add interactive JS (live file viewer, search, repo switching) if you want — say the word.
